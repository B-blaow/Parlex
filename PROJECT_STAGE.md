## Restore After Clean Clone

The repository intentionally does not store local build products, secrets, native engine checkouts, or model files.

Restore locally only when needed:

- `keystore.properties` and release keystore for signed builds.
- Android SDK / build tools, NDK `27.3.13750724`, CMake.
- `app/src/main/cpp/llama.cpp/` at tag `b9464`.
- Downloaded GGUF / LiteRT model files through the app or local tools.
- Gradle caches and `app/build/` outputs are regenerated by the build.

## Clean Folder Policy

These are intentionally ignored and safe to delete before a Windows reinstall:

- `.gradle/`
- `build/`
- `app/build/`
- `app/.cxx/`
- APK/AAB files
- `keystore.properties`
- local model files: `*.gguf`, `*.litertlm`, `*.tflite`, `*.task`, `*.safetensors`, `*.bin`
- local native engines: `app/src/main/cpp/llama.cpp/`, `app/src/main/cpp/whisper.cpp/`
- `diagnostics/`
